## What you'll install

1. **Python 3.11 or later** — the language we'll be using.
2. **Git** — version control.
3. **VS Code** — a free, beginner-friendly editor.
4. **VS Code extensions** — Python, Jupyter, GitLens.

That's it. We won't ask you to install anything paid or proprietary.

## After installing — verify

Open a terminal and run:

```bash
python --version    # should print 3.11.x or newer
git --version       # any 2.x is fine
code --version      # should print VS Code version info
```

If all three commands print versions, you're ready. Head back to [Week 1](../../curriculum/week-01-python-foundations/).

If any fail, retry the relevant section in your platform guide, or ask in [GitHub Discussions](https://github.com/CODE-CRUNCH-CLUB/C1-Code-Crunch-Convos/discussions).

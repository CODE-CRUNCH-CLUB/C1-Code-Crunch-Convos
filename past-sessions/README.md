# Past Sessions

Archived materials from prior cohorts. These are preserved for historical reference — they are **not** the current curriculum. Active learners should use the [main 15-week curriculum](../curriculum/) instead.

## Sessions

- **[SPRING 2025](SPRING-2025/)** — Original Code Crunch Convos series. 5 units focused on Python fundamentals and interview-prep data structures (sets, dictionaries, two pointers, linked lists, stacks & queues). PDF slide decks only.

## Why preserve these?

- **Historical context** — see how the program evolved.
- **Reference material** — some learners may find the interview-prep focused content useful as a supplement.
- **Acknowledging contributors** — past instructors and TAs put real work into these materials.

## Why not delete them?

Open source curricula benefit from being able to compare versions over time. Even outdated material can be useful for understanding pedagogical choices ("why did we change X?"). If a future cohort wants to revive an old unit's approach, it's right here.

## Contributing

We **do not** accept new contributions to past-session folders — they are frozen. Improvements should target the active [curriculum/](../curriculum/) tree.

# Week 3 — Exercises

Five short exercises to drill conditionals and loops. Each one ships as a
runnable `.py` file with a docstring spec and a skeleton. Read the
docstring, fill in the marked `TODO` sections, and run the file from your
terminal:

```bash
python exercise-01-fizzbuzz.py
```

You should be able to finish each exercise in **15–30 minutes**. If you
get stuck for more than that, re-read the matching section in the
lecture notes and try again. Working code beats elegant code at this
stage — get it running first, polish later.

## The Exercises

| # | File | What you will practise |
|---|------|------------------------|
| 1 | [exercise-01-fizzbuzz.py](./exercise-01-fizzbuzz.py) | `for`, `range`, `if`/`elif`/`else`, modulo |
| 2 | [exercise-02-sum-evens.py](./exercise-02-sum-evens.py) | Accumulator pattern, `range` with step |
| 3 | [exercise-03-password-checker.py](./exercise-03-password-checker.py) | `while True`, `break`, `any()`, guard clauses |
| 4 | [exercise-04-multiplication-table.py](./exercise-04-multiplication-table.py) | Nested `for` loops, `print` formatting |
| 5 | [exercise-05-find-prime.py](./exercise-05-find-prime.py) | `for` + `else`, early `return`/`break` |

## Suggested order

Do them in order. Each one builds on a pattern from the previous
exercise, and 1–2 are warm-ups for the harder 3–5.

## How to submit

If you are tracking your work in Git (highly recommended — see Week 1),
commit each exercise as you finish it:

```bash
git add exercises/exercise-01-fizzbuzz.py
git commit -m "Week 3: complete FizzBuzz exercise"
```

# Week 1 Exercises

Short, focused coding tasks that drill the concepts from the Week 1
lectures. Aim to finish each one in a single sitting. If you get stuck,
re-read the relevant lecture and try again — the goal is comfort, not
speed.

## How to Run an Exercise

1. Open a terminal.
2. Activate your Week 1 virtual environment (see Lecture 2).
3. From the `exercises/` folder, run:

   ```bash
   python exercise-01-hello-world.py
   ```

Each exercise file starts with a docstring describing the goal and the
expected output.

## Checking Your Work

There is no automated grader for Week 1. For each exercise, verify
yourself that:

- The script runs without errors.
- The printed output matches the docstring's "Expected output" section.
- Any answers you paste as comments are correct.

# Week 1 Resources

A curated list of free, high-quality resources that pair well with this
week's material. Read in the order listed for the smoothest ramp-up, or skip
to the ones that interest you. All entries are free of charge.

## Primary Reading

### 1. The Official Python Tutorial

- **Link:** <https://docs.python.org/3/tutorial/>
- **Why it's useful:** Written and maintained by the people who build Python.
  Authoritative, accurate, and surprisingly readable. If you ever wonder
  "what does the language actually say?", this is the answer.
- **Time to read this week:** 45–60 minutes (sections 1–3).

### 2. PEP 8 — Style Guide for Python Code

- **Link:** <https://peps.python.org/pep-0008/>
- **Why it's useful:** PEP 8 is the convention nearly every Python codebase
  follows for indentation, naming, and layout. Reading it now means the code
  you write this week already looks like professional Python.
- **Time to read this week:** 30 minutes (skim once, refer back often).

### 3. PEP 20 — The Zen of Python

- **Link:** <https://peps.python.org/pep-0020/>
- **Why it's useful:** Twenty short aphorisms that describe Python's design
  philosophy. Run `python -c "import this"` to see it in your terminal.
- **Time to read this week:** 5 minutes.

## Free Books

### 4. "Automate the Boring Stuff with Python" by Al Sweigart

- **Link:** <https://automatetheboringstuff.com/>
- **Why it's useful:** Practical, project-oriented, and free to read online.
  Chapter 1 ("Python Basics") covers the same ground as this week and is a
  great second pass.
- **Time to read this week:** 1 hour (chapter 1 only).

### 5. "Think Python 2e" by Allen Downey

- **Link:** <https://greenteapress.com/wp/think-python-2e/>
- **Why it's useful:** A computer scientist's introduction to Python. More
  theoretical than "Automate the Boring Stuff" — pairs nicely with it.
  Chapter 1 ("The way of the program") is a classic.
- **Time to read this week:** 30–45 minutes (chapter 1).

## Tutorials and Videos

### 6. Real Python Tutorials

- **Link:** <https://realpython.com/tutorials/basics/>
- **Why it's useful:** Bite-sized, well-illustrated articles that focus on a
  single concept at a time. The "Python Basics" track maps closely to this
  week's outline.
- **Time to read this week:** Pick 1–2 articles, ~20 minutes each.

### 7. freeCodeCamp Python Videos on YouTube

- **Link:** <https://www.youtube.com/c/Freecodecamp/search?query=python>
- **Why it's useful:** Long-form lecture-style videos. Try the
  "Learn Python — Full Course for Beginners" video for an alternative
  presentation of this week's content.
- **Time to watch this week:** 1–2 hours (the first 90 minutes cover Week 1
  ground).

## Git, GitHub, and VS Code

### 8. The Pro Git Book

- **Link:** <https://git-scm.com/book/en/v2>
- **Why it's useful:** The definitive Git reference, free online. Chapters 1
  and 2 cover everything you need this week.
- **Time to read this week:** 45 minutes (chapter 1 + section 2.1–2.3).

### 9. GitHub Docs — "Hello World"

- **Link:** <https://docs.github.com/en/get-started/quickstart/hello-world>
- **Why it's useful:** A guided walkthrough from GitHub itself for creating
  your first repository.
- **Time to read this week:** 20 minutes.

### 10. Visual Studio Code — Python Quick Start

- **Link:** <https://code.visualstudio.com/docs/python/python-tutorial>
- **Why it's useful:** Walks through installing the Python extension, running
  files, and using the integrated debugger — exactly what you'll do at the
  end of this week.
- **Time to read this week:** 25 minutes.

## How to Use These Resources

You do not need to read every link. Pick **one** primary source from the
"Free Books" or "Tutorials" sections and stick with it for the week. The
official docs and PEPs are reference material — bookmark them now and dip in
when something feels fuzzy. If you only have time for two resources this
week, pick the **Official Python Tutorial** (sections 1–3) and the
**Pro Git Book** (chapter 1).

# Week 7 — Challenges

Two longer problems that combine multiple OOP concepts. Treat them like
exam-style design exercises: read the spec, sketch your classes on paper,
*then* code. Each challenge has acceptance criteria and stretch goals.

| # | File                                  | Focus                                     |
|---|---------------------------------------|-------------------------------------------|
| 1 | `challenge-01-deck-of-cards.md`       | dunder methods, iteration, randomness     |
| 2 | `challenge-02-employee-hierarchy.md`  | inheritance, polymorphism, aggregation    |

These are graded against the **acceptance criteria** at the bottom of each
spec. If your code satisfies every bullet, you have nailed the challenge.

A reference solution is intentionally not provided — try, then compare with a
classmate's solution. Different correct designs are common and educational.

# Week 7 — Exercises

Five small, focused exercises. Each file is a runnable Python script — open
it, read the docstring, follow the `TODO` markers, and run it with:

```bash
python exercises/exercise-01-dog-class.py
```

The skeletons include type hints and example output in the docstring. Your
job is to fill in the gaps until the script behaves as documented.

| # | File                                  | Topics                                       |
|---|---------------------------------------|----------------------------------------------|
| 1 | `exercise-01-dog-class.py`            | classes, `__init__`, methods                 |
| 2 | `exercise-02-rectangle.py`            | `@property`, derived attributes              |
| 3 | `exercise-03-inheritance-shapes.py`   | inheritance, overriding, polymorphism        |
| 4 | `exercise-04-dataclass-user.py`       | `@dataclass`, auto-generated `__eq__`        |
| 5 | `exercise-05-bank-account.py`         | state, exceptions, `__repr__` vs `__str__`   |

Recommended order is 1 → 5. Each builds on a concept from the lecture notes:

- 1 and 2 → Lecture 01.
- 3 → Lecture 02.
- 4 and 5 → Lecture 03 (and tying everything together).

A solution is a script that runs to completion **with no errors** and prints
output that matches the docstring example. Try to do every exercise without
peeking at the next lecture — discomfort is part of learning.

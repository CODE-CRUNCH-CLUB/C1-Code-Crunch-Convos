# Week 13 — Resources

A curated list of the best free and paid resources for learning NumPy,
pandas, matplotlib, and the data-analysis workflow. You do not need to read
all of these — pick what fits your learning style and circle back to others
later in the program.

## Official Documentation

These are the source of truth. Bookmark them; you will return often.

- **pandas user guide** — <https://pandas.pydata.org/docs/>
  The canonical reference. The "User Guide" and "10 minutes to pandas" pages
  are ideal first reads. The API reference is where you look up arguments to
  specific functions.
- **NumPy documentation** — <https://numpy.org/doc/>
  Read "NumPy: the absolute basics for beginners" first. Then skim
  "Broadcasting" — it explains how arrays of different shapes combine.
- **matplotlib documentation** — <https://matplotlib.org/stable/>
  Has a deep gallery of plot examples you can copy and adapt. Start with
  "Quick start guide" and "Pyplot tutorial".
- **Jupyter project** — <https://jupyter.org/documentation>
  Notebook basics, keyboard shortcuts, magic commands.
- **seaborn** (optional, prettier defaults) — <https://seaborn.pydata.org/>

## Free Books and Long-form Tutorials

- **Jake VanderPlas — _Python Data Science Handbook_** (free, online):
  <https://jakevdp.github.io/PythonDataScienceHandbook/>
  Chapters 2 (NumPy), 3 (pandas), and 4 (matplotlib) cover everything in
  Week 13 and go further than we have room for. Highly recommended.
- **Wes McKinney — _Python for Data Analysis_, 3rd edition** (free online):
  <https://wesmckinney.com/book/>
  Wes is the original author of pandas. This book is the definitive guide.
  Chapters 4–10 align with this week.

## Free Datasets

You will need real datasets for the mini-project and homework. Start with
these catalogues:

- **Kaggle Datasets** — <https://www.kaggle.com/datasets>
  Tens of thousands of free datasets with discussion and example notebooks.
  Requires a free account.
- **data.gov** — <https://data.gov/>
  US government open data — census, weather, transportation, health.
- **Awesome Public Datasets** —
  <https://github.com/awesomedata/awesome-public-datasets>
  Curated GitHub list grouped by topic.
- **NYC Open Data** — <https://opendata.cityofnewyork.us/>
  Famous source of the NYC Taxi trips dataset.
- **Our World in Data** — <https://ourworldindata.org/>
  Cleaned CSVs on global development, climate, and health.
- **FiveThirtyEight data** —
  <https://github.com/fivethirtyeight/data>
  The data behind their published articles, mostly small CSVs.

## Video Courses (Free)

- **Corey Schafer — pandas tutorial series** on YouTube. Friendly and
  thorough — search for "Corey Schafer pandas".
- **Keith Galli — _Solving real-world data science tasks with pandas_** on
  YouTube. Project-based walkthroughs.

## Community

- **pandas tag on Stack Overflow** —
  <https://stackoverflow.com/questions/tagged/pandas>
  Almost any error message you hit has already been answered here.
- **r/learnpython** and **r/datascience** on Reddit for general questions.
- **pandas Discord** — links from the docs homepage.

## What to Read When You Get Stuck

1. The pandas API reference for the specific method.
2. Search the error message verbatim in a quote search ("error text").
3. The "10 minutes to pandas" guide at
   <https://pandas.pydata.org/docs/user_guide/10min.html>.
4. Ask your peers on the Code Crunch Convos community channel.

# Week 13 — Challenges

Two longer problems to deepen what you learned in the exercises. Each
challenge is open-ended — there is no single "correct" solution. Aim to
finish at least one; do both if you can.

| # | File | Topic |
|---|------|-------|
| 1 | [challenge-01-titanic-eda.md](./challenge-01-titanic-eda.md) | Exploratory data analysis on the Titanic dataset |
| 2 | [challenge-02-time-series.md](./challenge-02-time-series.md) | Resample a time series and plot a rolling mean |

## How to submit

- Create a new file per challenge in this folder named e.g.
  `solutions/challenge-01-titanic-eda.ipynb`.
- Include at least one markdown cell that states what the chart shows in
  plain English.
- Push your notebook to your fork; share the link in the community channel.

## Grading rubric (informal)

| Criterion              | Points |
|------------------------|-------:|
| Code runs end-to-end   | 4 |
| Cleaning is sensible   | 3 |
| At least one chart, labelled | 3 |
| Written interpretation | 2 |
| Style and readability  | 1 |
| **Total**              | **13** |

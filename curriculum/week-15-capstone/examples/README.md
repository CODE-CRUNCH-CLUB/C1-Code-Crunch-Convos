# Capstone Examples

These five examples are not the only capstones you can build — but each
is a realistic, week-sized project that fits one of the five tracks.
Browse them on Day 1 before you decide what to build. Each example
links to a detailed scope file with feature lists, tech recommendations,
and a milestone-by-milestone plan.

## Five more tiny ideas

If none of the above clicks, here are five smaller starter ideas — each
roughly capstone-sized:

- **Pomodoro CLI with stats** — a CLI that runs a Pomodoro timer, writes
  each session to SQLite, and shows weekly focus-time stats. Persists
  state. *Track: Automation or Web App.*
- **Recipe scaler** — paste in a recipe, scale it by a factor, save it
  to a JSON file, view past recipes. *Track: Web App.*
- **GitHub stars analyzer** — given a username, download their starred
  repos, cluster them by topic and language, and produce charts.
  *Track: Data.*
- **Sentiment of Hacker News headlines** — daily script that fetches
  the front page, scores sentiment per headline, and emails you a
  digest. *Track: Automation + light ML.*
- **Personal finance category classifier** — CSV of transactions in,
  trained classifier groups them into categories (food, rent,
  transport), CLI prints monthly summary. *Track: ML.*

## Reading order

If you have already chosen a track, jump to that example file. If you
are still deciding, skim all five examples — they take five minutes each
— and the right project usually announces itself.

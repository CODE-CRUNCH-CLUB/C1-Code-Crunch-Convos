# Week 9 — Challenges

Two larger problems that stitch together the lecture material. Unlike the
exercises, these are open-ended: you decide on the file layout, the
templates, and how to organise the code. There is a suggested approach in
each file, but feel free to deviate.

| # | File                                 | Skills exercised                                |
|---|--------------------------------------|-------------------------------------------------|
| 1 | `challenge-01-todo-app.md`           | Routes, forms, in-memory state, template loops  |
| 2 | `challenge-02-url-shortener.md`      | POST + redirect, slug generation, 404 handling  |

---

## Recommended order

Do them after lecture 3 and the four exercises, but **before** the
mini-project. Each takes about 2–4 hours.

## What to turn in

Nothing — these are practice. If you publish your code to GitHub, link it in
the bootcamp Discord and we will roast it gently.

## Difficulty signals

- **Smooth sailing** in under 2 hours? Try the stretch goals in each file.
- **Stuck for over an hour** on one spot? Skim the relevant lecture again,
  then read the Flask Quickstart section that matches your stuck point.
  Asking on the Discord is faster than spinning.
